A complete **13-phase AI-powered YouTube Shorts automation dashboard** for USA-focused faceless channels.

## ✨ Features

- ⚡ **One-Click Full Pipeline** — Run all 13 phases automatically
- 🎨 **Premium Dark UI** — Glassmorphism design with neon accents
- 🧠 **AI-Powered** — Google Gemini API (free tier)
- 💾 **Auto-Save** — All projects saved locally
- 📋 **Copy Outputs** — One-click copy for all generated content
- 📊 **Analytics Dashboard** — Track video performance
- 🆓 **100% Free** — No paid services required

## 🚀 13 Pipeline Phases

| # | Phase | Description |
|---|-------|-------------|
| 1 | 🔍 Trend Research | Discover top viral topics with scoring |
| 2 | 🎯 Content Strategy | Validate topic for USA audience |
| 3 | ✍️ Script Writer | Generate 30-60s Shorts script |
| 4 | 🎬 Visual Scenes | Scene breakdown with AI image prompts |
| 5 | 🎙️ Voiceover | Voice guide with TTS recommendations |
| 6-7 | 🎥 Video Plan | Editing blueprint for CapCut/DaVinci |
| 8 | 🖼️ Thumbnails | 5 thumbnail concepts with AI prompts |
| 9 | 🔎 SEO Optimizer | Titles, tags, hashtags, descriptions |
| 10 | ✅ Quality Control | Fact-check, copyright, compliance |
| 11 | 🚀 Publish Guide | Upload settings & optimal USA timing |
| 12 | 📈 Analytics | Performance tracking with charts |
| 13 | 🧠 Self-Improve | AI learns from past performance |

## 📦 Quick Start

1. **Get free API key**: [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Clone this repo:
   ```bash
   git clone https://github.com/ravimeen715-svg/Ravi-ai.git
   ```
3. Open `index.html` in your browser
4. Enter your Gemini API key
5. Click **"Run Full Pipeline"** 🎉

## 🛠️ Tech Stack

- **Frontend**: HTML + CSS + JavaScript (no frameworks)
- **AI Engine**: Google Gemini 2.0 Flash (free tier)
- **Storage**: Browser LocalStorage
- **Design**: Custom dark glassmorphism theme
